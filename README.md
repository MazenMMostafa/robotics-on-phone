# Robotics On Phone

Block coding for Arduino Uno, Arduino Nano, and ESP32 — right from your phone.

Build projects using visual blocks, generate Arduino code, then upload firmware directly over USB OTG.

## Features

- Visual block programming with Google Blockly
- Support for Arduino Uno, Arduino Nano, and ESP32
- Automatic Arduino C++ code generation
- One-click firmware compilation and upload via USB OTG
- Built-in serial terminal for debugging
- Project management with auto-save
