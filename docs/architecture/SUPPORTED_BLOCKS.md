## Logic (6)
| Block Type | Fields/Inputs | Purpose | Output |
|-----------|--------------|---------|--------|
| `controls_if` | IF0..N, DO0..N, ELSE, mutation | Conditional | `if/else if/else` block |
| `logic_compare` | `OP` (EQ/NEQ/LT/GT/LTE/GTE), A, B | Comparison | `(A == B)` etc. |
| `logic_operation` | `OP` (AND/OR), A, B | Boolean op | `(A && B)` / `(A \|\| B)` |
| `logic_negate` | BOOL input | Negation | `!(BOOL)` |
| `logic_boolean` | `BOOL` (TRUE/FALSE) | Literal | `true` / `false` |
| `logic_null` | (none) | Null value | `false` |

## Loops (2)

| `controls_whileUntil` | WHILE input, DO input, mutation | While/until | `while (COND)` / `while (!COND)` |

## Math (3)
